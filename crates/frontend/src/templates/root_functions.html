Root Functions and Their Call Trees

A Root Function is the entry point to a series of function calls in a binary. These roots serve as the head of their respective call graphs, which show the chain of function invocations starting from that point.

A Call Graph is a graphical representation of the relationships between functions in a binary. Understanding these graphs helps you analyze control flow, detect vulnerabilities, and optimize performance.
